# 刀客塔的办公室 Dokta's Office
> 一款基于 Mirai 框架的 QQ 机器人应用，其灵感来源于游戏明日方舟。

该插件基于 Mirai-console 开发，请确保您机器人使用的是 [Mirai]([https://github.com/mamoe/mirai]) 框架。

**本应用将于不远的未来完全开源，敬请期待。**

## 预计支持的功能

- [ ] 公开招募词条分析
- [ ] 干员档案表单生成
- [ ] 干员招募寻访系统
- [ ] 读取材料掉落矩阵
- [ ] 获取泰拉大陆新闻
- [ ] 查询博士社交资料
- [ ] 录入你的入职信息
- [ ] 整理本舰的办公室
- [ ] 返回你的办公室
